# Pneumonia Detection using Ensemble Learning

## 📌 Project Overview
This project detects pneumonia from chest X-ray images using a combination of deep learning and machine learning models.

## 🚀 Technologies Used
- Python
- TensorFlow / Keras (VGG16)
- Scikit-learn
- LightGBM
- OpenCV

## 🧠 Methodology
- Feature extraction using VGG16
- Ensemble classifiers:
  - KNN
  - Naive Bayes
  - Decision Tree
  - Random Forest
  - LightGBM
- Voting Classifier for final prediction

## 📊 Results
- Accuracy: 96%
- Improved performance over individual models

## 📁 Dataset
Kaggle Chest X-ray Dataset

## 📷 Outputs
- Confusion Matrix
- ROC Curve
- Accuracy Comparison
